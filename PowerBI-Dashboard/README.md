
# Power BI Dashboard

Interactive Power BI dashboard developed for Olist E-Commerce analysis.

## Features

- Dynamic KPI tracking
- Revenue analysis
- Payment distribution
- Delivery performance
- Customer review insights
- Interactive filters and slicers
